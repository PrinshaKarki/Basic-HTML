What is HTML?

HTML, which stands for Hypertext Markup Language, is a pretty simple to understand. It consists of different elements which we use to structure or create a web page.

What is a Element?

HTML element always start with a opning tag, having different names. THe name of a element are bounded by opening and closing angle bracket.

As the opening tag, closing tag also consists of two angle brackets surrounding the name of an element. But it has a slash in front of the name of a element to differentiate between opning and closing tag.

Evervthing comming between opening and closing tag is known as content.

Not all the element have a content. In other words, it is also known as empty tag. They only have opning tag that does not have any content.

For Example: img and input tag.

What are HTML Attributes?

Attributes are the extra information about the element that will not appear in the content.

Block-level and inline HTML elements

By default, an element can be either block-level or an inline element.
Block-level elements are the elements that always start on a new line and take all the space.
Inline elements are the elements that do not start on a new line and it only take up as much space as necessary.
